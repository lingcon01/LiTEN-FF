LiTEN-FF is a MLIP model based on the **LiTEN** framework. It is trained and fine-tuned on the **NablaDFT** and **SPICE** datasets, and is designed for a wide range of applications, including **conformational optimization**, **molecular dynamics simulations**, and **conformer searching**. The detailed usage instructions can be found at [LiTEN GitHub](https://github.com/lingcon01/LiTEN/tree/master).
